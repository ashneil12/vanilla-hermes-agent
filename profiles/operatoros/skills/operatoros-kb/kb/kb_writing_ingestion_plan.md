# KB Ingestion Plan (Writing)

## Active vs archive
**Active (Tier 1):** Copy & Content Playbook (Defluffed)  
**Archive (Tier 3):** full transcripts + long explainers + big swipe banks

## Chunking (RAG-friendly)
Chunk by job-to-be-done:
- Hooks
- Landing pages
- Emails
- Newsletters
- Editing
- Positioning
- Repurposing

Each chunk should include: when to use + rule/test + template + checklist.

## Retrieval routing
- “Write X” → templates + checklists
- “Why isn’t X converting” → diagnostics/tests
- “Make it sound like Y” → swipe bank + tone notes
